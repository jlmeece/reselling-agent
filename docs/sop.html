§3 — Status Reference
PENDING
Discovered, not yet scored
→ Run Research
SCORED
Tier 1 result — needs your call
→ Approve or skip in the sheet
WATCH
Tier 2 — promising, not ready
→ Re-scored weekly automatically
APPROVED
You greenlighted it
→ Run Daily to promote to READY
READY
Copy done, stock confirmed
→ Export CSV → list on eBay
LISTED
CSV uploaded to eBay
→ Paste live eBay URL into col Q
ACTIVE
Live on eBay, monitored
→ Nothing — agent watches it
PAUSED_OOS
Costco out of stock
→ Pause your eBay listing
PAUSED_MARGIN
Margin dropped below threshold
→ Lower price or end listing
PAUSED_DEMAND
Tier 3 — not viable
→ Re-eval in 30 days
PAUSED_SEASONAL
Wrong season
→ Agent re-evals on re-eval date
